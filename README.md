## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/archakNath/BusBondhu.git
cd BusBondhu
```

### 2. Setup Backend
```bash
npm install
npm run dev
```
Runs the backend server at http://localhost:5000/

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
Runs the frontend app at http://localhost:5173/
